# Snake Game

A simple browser-based Snake game built with HTML, CSS, and JavaScript.

## Features

- Classic snake movement using arrow keys
- Food generation and eating mechanic
- Score tracking
- High score saved in localStorage
- Restart and start game modals

## Files

- `index.html` — the game markup
- `style.css` — the visual styles
- `theme.css` — theme variables and colors
- `script.js` — game logic and controls

## How to run

1. Open `index.html` in a web browser.
2. Click the **Start** button.
3. Use the arrow keys to move the snake.

## Controls

- `ArrowUp` — move up
- `ArrowDown` — move down
- `ArrowLeft` — move left
- `ArrowRight` — move right

## Notes

- The game ends if the snake hits a wall or itself.
- The current high score is saved in the browser using `localStorage`.
